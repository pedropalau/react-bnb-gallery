## Install

You can install [react-bnb-gallery](https://www.npmjs.com/package/react-bnb-gallery) from [npm](https://www.npmjs.com/).

```bash
# If you are using npm
npm install --save react-bnb-gallery

# If you prefer yarn
yarn add react-bnb-gallery
```
